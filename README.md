# 🎮 게임 아케이드

브라우저에서 바로 즐기는 실시간 멀티플레이 게임 모음입니다.  
설치 없이 링크 하나로 접속하고, 방 코드만 공유하면 친구와 바로 플레이할 수 있습니다.

**▶️ 플레이:** https://eunseong965.github.io/Blind-Formula/

---

## 게임 목록

### 🃏 블라인드 수식전 · 온라인 2인
상대의 카드를 모른 채 수식을 완성해 점수를 겨루는 2인 두뇌 대전입니다.

- 카드를 번갈아 놓아 `(숫자)(연산)(숫자)(연산)(숫자) = ?` 형태의 수식을 완성합니다.
- 상대가 놓은 카드는 보이지 않고, **수식의 결과값만** 공개됩니다.
- 4라운드 점수 합산으로 승부합니다.
- 패자는 **러시안 룰렛**을 진행합니다. 약실 6칸·실탄 3발로 시작하며, 살아남을수록 위험이 커집니다.

---

### ⚔️ 정복 오목 · 온라인 4인
영토를 넓히고 전투로 빼앗으며 5목을 노리는 4인 영토 배틀로얄입니다.

- 8×8 판에서 말을 움직여 빈 칸을 점령합니다.
- 말은 상하좌우 1~4칸 이동할 수 있으며, 경로를 직접 그리듯 자유롭게 꺾을 수 있습니다.
- 상대 말과 부딪히면 1~4 카드 전투가 발생합니다.
- 전투에서 이기면 상대 땅을 빼앗아 한 바퀴 동안 고정합니다.
- 패배한 말은 자기 땅으로 후퇴하며, 후퇴할 곳이 없으면 탈락합니다.
- 승리 조건은 **내 땅 5목**, 최후 생존, 또는 빈 칸 소진 시 최다 영토입니다.
- 말 배치와 이동에 애니메이션이 적용되어 점령과 이동 흐름을 더 직관적으로 볼 수 있습니다.

---

### 🎭 도플갱어 게임 · 온라인 5인
5명 중 1명의 가짜를 찾아내는 채팅형 프로필 추리게임입니다.

- 5명 중 1명은 **도플갱어**입니다.
- 도플갱어는 네 학생의 정보인 **성별·나이·보석·시크릿 넘버**를 뒤섞어 만들어진 가짜 프로필을 가집니다.
- 도플갱어의 가짜 프로필은 최소 2명 이상의 학생 정보가 섞이도록 생성됩니다.
- 학생들은 채팅으로 단서를 모아 **도플갱어가 누구인지 + 도플갱어의 가짜 프로필 4가지**를 맞혀야 합니다.
- 학생은 라운드당 **2번**까지 추리를 제출할 수 있습니다.
- 추리 결과는 **5개 중 몇 개가 맞았는지**만 알려줍니다.
- 단, **시크릿 넘버는 1~64 범위**이며, 틀렸을 경우 입력값보다 높은지 낮은지 업/다운 힌트를 받습니다.
- 도플갱어는 다른 사람 이름으로 **둔갑**해 채팅을 교란할 수 있습니다.
- 둔갑당한 당사자는 위조 발언을 감지할 수 있지만, 다른 사람들은 그것이 진짜 해명인지 거짓말인지 판단해야 합니다.
- 도플갱어는 학생의 프로필 5개를 모두 맞히면 해당 학생을 **도플갱어화**시킵니다.
- 도플갱어화된 학생은 탈락하지 않고 채팅에는 참여할 수 있지만, 더 이상 추리는 제출할 수 없습니다.
- 투표는 없습니다. 오직 추리만이 진실에 닿습니다.

## 기술 스택

- **바닐라 HTML / CSS / JavaScript** — 프레임워크 없이 구현
- **Firebase Realtime Database** — 실시간 멀티플레이 동기화
- **GitHub Pages** — 정적 호스팅
- 게임 허브는 **iframe 격리** 방식으로 각 게임을 독립 실행

## 플레이 방법

1. 위 링크에 접속해 게임을 선택합니다.
2. **방 만들기**로 방을 생성하고, 표시된 **방 코드**를 친구에게 공유합니다.
3. 친구는 같은 화면에서 **방 코드로 참가**합니다.
4. 인원이 모이면 방장이 게임을 시작합니다.
