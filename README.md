# 🎮 게임 아케이드

브라우저에서 바로 즐기는 실시간 멀티플레이 게임 모음.
설치 없이 링크 하나로, 방 코드만 공유하면 친구와 바로 플레이할 수 있습니다.

**▶️ 플레이: https://eunseong965.github.io/Blind-Formula/**

---

## 게임 목록

### 🃏 블라인드 수식전 · 온라인 2인
상대의 카드를 모른 채 수식을 완성해 점수를 겨루는 두뇌 대전.

- 카드를 번갈아 놓아 `(숫자)(연산)(숫자)(연산)(숫자) = ?` 수식을 완성
- 상대가 놓은 카드는 보이지 않고 **결과 값만** 공개되는 블라인드 방식
- 4라운드 점수 합으로 승부
- 패자는 **러시안 룰렛** — 약실 6칸·실탄 3발로 시작해 살아남을수록 확률이 오른다

### ⚔️ 정복 오목 · 온라인 4인
영토를 넓히고 전투로 빼앗으며 5목을 노리는 영토 배틀로얄.

- 8×8 판에서 말을 움직여 빈 칸을 점령
- 상하좌우 1~4칸, 경로를 직접 그리며 이동 (자유롭게 꺾기 가능)
- 상대 말과 부딪히면 1~4 카드 전투, 이기면 상대 땅을 빼앗아 한 바퀴 고정
- 패배 시 자기 땅으로 후퇴, 갈 곳 없으면 탈락
- 승리: **내 땅 5목** / 최후 생존 / 빈 칸 소진 시 최다 영토

### 🎭 도플갱어 게임 · 온라인 5인
5명 중 1명의 가짜를 찾아내는 채팅 추리게임.

- 5명 중 1명은 **도플갱어** — 네 학생의 정보(성별·나이·보석·시크릿넘버)를 뒤섞은 가짜
- 도플갱어는 매번 한 사람으로 **둔갑**해 채팅을 교란한다
- 학생들은 실시간 채팅으로 단서를 모아 **도플갱어의 정체와 가짜 프로필 5가지**를 추리
- 도플갱어는 학생의 프로필을 맞혀 하나씩 감염시킨다
- 투표는 없다. 오직 추리만이 진실에 닿는다

## 기술 스택

- **바닐라 HTML / CSS / JavaScript** — 프레임워크 없이 구현
- **Firebase Realtime Database** — 실시간 멀티플레이 동기화
- **GitHub Pages** — 정적 호스팅
- 게임 허브는 **iframe 격리** 방식으로 각 게임을 독립 실행

## 파일 구조

| 파일 | 설명 |
|------|------|
| `index.html` | 게임 아케이드 허브 (게임 선택 화면) |
| `blind.html` | 블라인드 수식전 (온라인 2인) |
| `territory-online.html` | 정복 오목 (온라인 4인) |
| `doppelganger.html` | 도플갱어 게임 (온라인 5인) |

## 플레이 방법

1. 위 링크에 접속해 게임 선택
2. **방 만들기**로 방을 생성하고, 표시된 **방 코드**를 친구에게 공유
3. 친구는 같은 화면에서 **방 코드로 참가**
4. 인원이 모이면 방장이 게임 시작
