# 🎮 게임 아케이드

브라우저에서 바로 즐기는 미니게임 모음. 한 입구에서 여러 게임을 골라 플레이할 수 있습니다.

**▶️ 플레이: https://eunseong965.github.io/Blind-Formula/**

---

## 게임 목록

### 🃏 블라인드 수식전
상대의 카드를 모른 채 수식을 완성해 점수를 겨루는 온라인 2인 두뇌 대전.

- 각자 카드 10장으로 `(숫자)(연산)(숫자)(연산)(숫자) = ?` 수식을 번갈아 채움
- 상대가 놓은 카드는 보이지 않고, 결과 값만 공개되는 **블라인드** 방식
- 4라운드 동안 자기 라운드 점수를 합산해 승부
- 패자는 **러시안 룰렛** — 약실 6칸 중 실탄 3발로 시작, 살아남을수록 확률 상승
- 방 코드로 1:1 매칭하는 실시간 온라인 대전

### ⚔️ 정복 오목
영토를 넓히고 전투로 빼앗으며 5목을 노리는 4인 영토 배틀로얄.

- 8×8 판에서 각자 말 1개로 시작, 빈 칸에 멈추면 그 칸이 내 땅
- 상하좌우로 1~4칸, 경로를 직접 그리며 이동 (자유롭게 꺾기 가능)
- 상대 말과 부딪히면 1~4 카드 전투, 이기면 상대 땅 하나를 빼앗아 한 바퀴 동안 고정
- 패배 시 자기 땅으로 후퇴 (갈 곳 없으면 탈락)
- 승리 조건: **내 땅 5목** / 최후 생존 / 빈 칸 소진 시 최다 영토
- 방 코드로 매칭하는 실시간 온라인 4인 대전

## 기술 스택

- **순수 HTML / CSS / JavaScript** — 프레임워크 없이 바닐라로 구현
- **Firebase Realtime Database** — 실시간 멀티플레이 동기화
- **GitHub Pages** — 정적 호스팅
- 게임 허브는 **iframe 격리** 방식으로 각 게임을 독립 실행

## 파일 구조

| 파일 | 설명 |
|------|------|
| `index.html` | 게임 아케이드 허브 (게임 선택 화면) |
| `blind.html` | 블라인드 수식전 (온라인 2인) |
| `territory-online.html` | 정복 오목 (온라인 4인) |

## 플레이 방법

1. 위 링크에 접속
2. 플레이할 게임 선택
3. **방 만들기**로 방을 생성하고, 표시된 **방 코드**를 친구에게 공유
4. 친구는 같은 화면에서 **방 코드로 참가**
5. 인원이 모이면 방장이 게임 시작
